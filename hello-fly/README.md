##初見Flyctl

- 安裝
    - ```iwr https://fly.io/install.ps1 -useb | iex```
    - [教學](https://fly.io/docs/hands-on/install-flyctl/)
- 官方部屬教學
    - [教學](https://fly.io/docs/languages-and-frameworks/python/)
- 登入
    - ```flyctl auth login```
- 建立 App
    - ```flyctl launch```
- 部署 App 到 Fly.io
    - ```flyctl deploy```
- 查看部署狀態
    - ```flyctl status```
- 開啟 App
    - ```flyctl open```
- 查看 IP
    - ```flyctl ips list```
